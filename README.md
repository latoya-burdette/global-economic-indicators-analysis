# Global Economic Indicators Analysis (2005–2025)

This project explores global economic indicators from 2005 to 2025 using R. The goal was to look at how key measures like GDP, GDP growth, inflation, unemployment, and GDP per capita have changed over time and how those trends relate to major global economic events.

The analysis focuses on identifying long-term patterns in global economic activity and highlighting periods where significant economic disruptions affected growth trends.

The project includes visualizations of:

• Global GDP over time
• Global GDP growth trends
• Global inflation rates
• Global unemployment rates
• Global GDP per capita

These visualizations were created using R with the tidyverse and ggplot2. The analysis itself was written in R Markdown and exported as an HTML report.

The repository includes the R Markdown file used for the analysis, the generated HTML report, and the datasets used to produce the visualizations.
